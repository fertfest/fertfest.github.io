类型推断

let helloWorld = "Hello World";

可以推断出helloWorld的类型为string。

定义类型

自动推断：

const user = {
  name: "Hayes", // string
  id: 0, // number
};

显式地描述上面的对象的“形状”：

interface User {
  name: string;
  id: number;
}

声明遵循接口“形状”的对象：

const user: User = {
  name: "Hayes",
  id: 0,
};

使用类声明接口：

interface User {
  name: string;
  id: number;
}
 
class UserAccount {
  name: string;
  id: number;
 
  constructor(name: string, id: number) {
    this.name = name;
    this.id = id;
  }
}
 
const user: User = new UserAccount("Murphy", 1);

使用接口注明函数参数和返回值：

function deleteUser(user: User) {
  // ...
}
 
function getAdminUser(): User {
  //...
}

除了JS中的 boolean, bigint, null, number, string, symbol, 和 undefined类型，TS扩展了更多类型：

  • any
  • unknow
  • never
  • void

复合类型

Union

使用union可以声明一个类型可能是多个类型中的一个。

type MyBool = true | false;
type WindowStates = "open" | "closed" | "minimized";
type LockStates = "locked" | "unlocked";
type PositiveOddNumbersUnderTen = 1 | 3 | 5 | 7 | 9;

union也提供了一种处理不同类型的方式：

function getLength(obj: string | string[]) {
  return obj.length;
}

使用typeof查询一个变量的类型:

Type Predicate
string typeof s === "string"
number typeof n === "number"
boolean typeof b === "boolean"
undefined typeof undefined === "undefined"
function typeof f === "function"
array Array.isArray(a)

泛型(Generics)

不带泛型的数组可以包含任何值。带泛型的数组可以限定其元素的类型：

type StringArray = Array<string>;
type NumberArray = Array<number>;
type ObjectWithNameArray = Array<{ name: string }>;

使用泛型声明自己的类型：

interface Backpack<Type> {
  add: (obj: Type) => void;
  get: () => Type;
}

// 告诉TS有一个常量叫做'backpack'，而不关心常量从哪来
declare const backpack: Backpack<string>;

// object是一个字符串
const object = backpack.get();

结构化类型系统

在一个结构化类型系统中，如果两个对象有着相同的行状态，可以说他们有相同的类型：

interface Point {
  x: number;
  y: number;
}
 
function logPoint(p: Point) {
  console.log(`${p.x}, ${p.y}`);
}
 
// logs "12, 26"
const point = { x: 12, y: 26 };
logPoint(point);

形状匹配只要求对象字段被要求的部分匹配即可：

const point3 = { x: 12, y: 26, z: 89 };
logPoint(point3); // logs "12, 26" 
 
const rect = { x: 33, y: 3, width: 30, height: 80 };
logPoint(rect); // logs "33, 3"
 
const color = { hex: "#187ABF" };
logPoint(color); // fail!!!

对于类和对象如何满足形状要求没有区别：

class VirtualPoint {
  x: number;
  y: number;
 
  constructor(x: number, y: number) {
    this.x = x;
    this.y = y;
  }
}
 
const newVPoint = new VirtualPoint(13, 56);
logPoint(newVPoint); // logs "13, 56"
